# Paralell-Merge-Sort
Merge Sort using Threads in C++

## Build Configurations

### Serial

       make serial
       ./MST.out {file-path}

       make serial-debug (For Printing the output to console)

### Multi-threaded

       make paralell
       ./MST.out {file-path} {min-elements} {max-threads}

       make paralell-debug (For Printing the output to console)
