# Paralell-Merge-Sort
Merge Sort using Threads in C++

## Build Configurations
 
### Serial

       make serial
       ./MST.out {file-path}

### Multi-threaded
      
       make paralell
       ./MST.out {file-path} {min-elements} {max-threads}
   
   
#### Ref:

ThreadPool Implementation was taken from https://github.com/progschj/ThreadPool
